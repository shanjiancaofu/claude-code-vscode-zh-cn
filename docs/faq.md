# 常见问题

## 插件更新后中文消失了？

Claude Code 更新通常会创建新版本目录。重新执行 dry-run 和 apply 即可；若新文案未覆盖，请提交版本号、截图和英文原文。

## 为什么恢复被拒绝？

当前文件哈希与补丁后哈希、原始哈希都不一致，说明文件被更新或被其他工具修改。工具会停止恢复，避免覆盖未知改动。

## 会读取或上传项目代码吗？

不会。本工具只读取三个 Claude Code 扩展文件，不访问工作区代码，也没有遥测或上传逻辑。

## 卸载工具会自动恢复吗？

不会。VS Code 没有可靠的“仅卸载时”钩子。请先执行 `Restore Before Uninstall` 或 `npm run restore`。
